# StellarLens VS Code Extension — Placeholder

This package is reserved for the StellarLens VS Code extension, which
will be implemented in **Phase 5** of the project roadmap.

When complete, this extension will provide three core capabilities,
all powered by the StellarLens SDK:

- **Hover-to-inspect** — hover over a contract address to see all
  available functions and their parameter types inline
- **Inline simulation** — select a transaction in the editor and
  simulate it without leaving VS Code
- **Call template generation** — right-click any contract function
  to insert a ready-to-use TypeScript SDK snippet

See the project roadmap for full details. Period.
